# CTF隐写从0到1教程

这是一个面向 CTF 初学者的隐写学习仓库。项目重点不是堆工具名，而是帮助你建立一条从文件类型识别、异常定位、载荷提取到二次解码/解密的稳定做题流程。

仓库同时提供章节讲义、实操练习、辅助脚本和完整版导出文档，适合自学、训练营整理和赛前速查。

## 项目定位

- 从 0 到 1 学习 CTF 隐写的入门教程
- 以“排查流程”而不是“工具清单”为核心组织内容
- 配套 17 道可直接动手做的练习题
- 提供 PowerShell 与 Node 脚本辅助复现实操样本

## 适合谁

- 刚开始接触 CTF Misc / 隐写方向
- 知道一些工具名，但不会组织排查顺序
- 能看懂命令输出，但不确定下一步该做什么
- 想把“会做一道题”升级成“知道一类题怎么排查”

## 实操练习

`practice/` 目前收录 17 道样本题，覆盖以下主题：

- 文件尾拼接
- OpenSSL `Salted__` 二层密文
- PNG alpha / 低对比度 / 调色板 / LSB
- GIF 拆帧与二维码闪现
- 音频摩斯与频谱写字
- 视频抽帧
- JPEG XMP / 元数据 / 组合密文
- DNS 流量隐写入门

建议按 [`practice/README.md`](./practice/README.md) 中给出的顺序做题；`challenge.md` -> `hint.md` -> `solution.md` 是默认使用方式。

## 辅助脚本

仓库内置了几类辅助脚本：

- [`scripts/Extract-AppendedZip.ps1`](./scripts/Extract-AppendedZip.ps1)：提取 JPEG / PNG / GIF 等文件尾部拼接的 ZIP
- [`scripts/Decrypt-OpenSslSalted.ps1`](./scripts/Decrypt-OpenSslSalted.ps1)：解 OpenSSL `Salted__` 格式的 AES-CBC Base64 密文
- [`scripts/Generate-PracticeSamples.ps1`](./scripts/Generate-PracticeSamples.ps1)：重新生成部分练习附件
- [`scripts/media_builder/`](./scripts/media_builder/)：为高级媒体题生成 GIF / WAV / QR / PCAP / WebM 等样本

更具体的依赖和示例命令见 [`scripts/README.md`](./scripts/README.md)。

如果你准备继续扩展这个仓库，建议先阅读：

- [`docs/文档规划.md`](./docs/文档规划.md)：定义首页、章节、练习区、脚本区各自职责
- [`docs/README写作规范.md`](./docs/README写作规范.md)：统一 README 结构、标题层级、链接风格与更新规则

当前策略是：

- 顶层 `README.md` 只做仓库导航，不承载整本讲义正文
- 知识内容放在章节文档里
- 剧透与答案集中放在 `practice/` 内部，不放在首页
- 项目级规范统一收敛到 `docs/` 目录

## 推送到 GitHub 前建议

- 提交前先检查相对链接是否都能正常打开
- 不要把 Office 临时文件、`node_modules` 等本地内容提交到仓库
- `practice/writeups-index.md` 已包含答案，公开仓库时要保留剧透提示
- 当前二进制文件体积较小，可以直接提交；若后续加入更大的媒体附件，再考虑 Git LFS

## 一句话总纲

隐写题真正考的不是“你会多少工具”，而是“你能否把文件格式、异常特征、密码学线索和脚本验证串成一条稳定流程”。
